
### 🎯 项目目标

使用 **PySide6** 开发一款名为 **“伯恩斯情绪工具集”** 的桌面应用程序，提供多个情绪管理工具，采用模块化设计，支持未来扩展。

---

### 🧱 核心架构要求

1. **主窗口（MainWindow）**
   - 采用**卡片式布局**（Card Layout），展示所有可用工具。
   - 每个卡片包含：
     - 工具名称（`name`）
     - 工具简介（`description`）
   - 卡片点击后，打开对应工具的独立子窗口（非模态或模态视具体功能而定）。
   - 主窗口保持运行，允许多个工具窗口同时打开。

2. **工具数据源**
   - 所有工具信息从 `data/tools.json` 文件中读取。
   - JSON 格式示例：
     ```json
     [
       {
         "name": "消极思维日志",
         "description": "记录并分析你的自动负性思维，帮助识别认知扭曲并建立理性回应。"
       },
       {
         "name": "消极思维计数器",
         "description": "统计每日消极思维出现次数，辅助追踪情绪变化趋势。"
       }
     ]
     ```
   - 程序启动时动态加载此文件，生成对应数量的卡片。

3. **工具窗口机制**
   - 每个工具在独立的 `QDialog` 或 `QWidget` 窗口中实现。
   - 窗口标题为工具名称。
   - 支持关闭后返回主窗口继续操作其他工具。

---

### 🛠️ 工具功能详情

#### ✅ 工具一：消极思维日志（Negative Thought Journal）

- **主界面布局**：
  - 垂直布局，上方为已保存的日志条目（卡片形式展示）。
  - 下方为“新建日志”按钮。

- **日志条目卡片**：
  - 显示摘要信息（如：日期、情景/下意识思维片段等）。
  - 可点击进入编辑模式（后续可扩展）。

- **“新建日志”流程**：
  1. 点击“新建”按钮，弹出选择对话框。
  2. 选择模板类型：
     - 三栏模板：`下意识思维`、`认知扭曲`、`理性回应`
     - 六栏模板：`情景`、`情绪`、`下意识思维`、`认知扭曲`、`理性回应`、`结果`
  3. 确认后进入编辑界面。

- **编辑界面布局**：
  - 横向两行布局：
    - 第一行：显示各栏标题（静态 QLabel）。
    - 第二行：对应文本输入框（QTextEdit 或 QLineEdit，建议用 QTextEdit 支持多行）。
  - 底部包含“保存”和“取消”按钮。

- **数据持久化**：
  - 保存路径：`data/消极思维日志.json`
  - 保存后，主界面刷新显示新增条目。

---

#### ✅ 工具二：消极思维计数器（Negative Thought Counter）

- **核心功能**：
  - 显示一个可编辑的数字计数器（`QSpinBox` 或 `QLineEdit` + 按钮）。
  - 提供“+”（增加）、“-”（减少）按钮，每次变化为 1。
  - 用户也可直接输入数值。

- **保存功能**：
  - 点击“保存”按钮，将当前数值与**今日日期**（YYYY-MM-DD）保存至：
    - 路径：`data/消极思维计数.json`
    - 文件格式（JSON 对象，以日期为键）：
      ```json
      {
        "2025-04-05": 7,
        "2025-04-06": 5
      }
      ```
  - 若当日已有记录，可选择覆盖或提示确认。

- **统计功能**：
  - 点击“统计”按钮，弹出统计窗口。
  - 显示折线图（使用 `PyQtGraph` 或 `matplotlib` + `FigureCanvas`）。
  - X轴：日期；Y轴：次数。
  - 提供时间范围选择：
    - 近7天
    - 近30天
    - 近90天
    - 近180天
    - 近365天
  - 支持动态切换时间范围并刷新图表。
